# CustomSaveDataUtilities

This is a utilities mod for *The Long Dark* enabling custom save data on gear items.

## Installation

1. If you haven't done so already, install MelonLoader by downloading and running [MelonLoader.Installer.exe](https://github.com/HerpDerpinstine/MelonLoader/releases/latest/download/MelonLoader.Installer.exe)
2. Download the latest version of `CustomSaveDataUtilities.dll` from the [releases page](https://github.com/ds5678/CustomSaveDataUtilities/releases).
3. Move `CustomSaveDataUtilities.dll` into the Mods folder in your TLD install directory
